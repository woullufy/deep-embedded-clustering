# Deep Embedded Clustering in PyTorch

PyTorch implementations of **Autoencoders**, **DEC**, and **IDEC** for  
unsupervised representation learning and clustering in latent space.

---

## Features
- Autoencoder & Denoising Autoencoder (AE/DAE)
- Deep Embedded Clustering (DEC)
- Improved DEC with reconstruction (IDEC)
- K-means initialization
- Plotting utilities for losses and reconstructions

---

## Project Structure
```text
models/       # AE, DAE, DEC, IDEC
utils/        # losses, plotting, clustering helpers
data/         # dataset loading and transforms
notebooks/    # example training workflows
```
